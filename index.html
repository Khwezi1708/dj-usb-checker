DJ USB confidence before every set.

Pioneer USB Checker validates FAT32/MBR compatibility, flags risky drives, and can watch USB insert events in the background. Built for fast pre-gig checks on macOS.

View on GitHub Quick Start

Status Model

  • PASS - healthy and correctly formatted.
  • WARN - usable with caveats.
  • FAIL - not Pioneer-safe.

Strict profile checks FAT32 + MBR + single partition. FAT32 is chosen because it is the most universal format across Pioneer decks.

Why DJs use it

Strict Compatibility Rules

Enforces Pioneer-safe filesystem and partition rules so you catch format issues before showtime.

Fast Health Signal

Checks mount/accessibility and optional write-read probe for practical readiness.

USB Insert Automation

Watch mode + LaunchAgent setup allows background auto-checks and notifications.

How it works

STEP 1

Detect

Reads external disks via macOS `diskutil`.

STEP 2

Normalize

Builds a stable device model from disk + partition metadata.

STEP 3

Evaluate

Applies hard rules and warning checks against your USB.

STEP 4

Report

Returns clear PASS/WARN/FAIL output in CLI/UI/notifications.

Quick install

git clone https://github.com/khwezi1708/dj-usb-checker.git
cd usb-checker
python3 -m venv .venv
source .venv/bin/activate
pip install -e .

usb-checker scan
usb-checker check disk6
usb-checker watch --poll-interval-sec 2 --debounce-sec 1.5
